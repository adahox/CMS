# Modulos

Todas as vezes que eu falar sobre criar um novo model, controller ou modulo, você deve entender que estou falando da estrutura toda como Controller -> Service -> Repository -> Model -> Migration.

## When to Use

- Quando for falado em implementações novas.
- Essa skill é boa para ajudar na velocidade do desenvolvimento

## Instructions

- Valida oque foi pedido
- envolve novos modulos ou coisas novas exceto campos específicos ou componentes.
- pergunte se realmente é para criar estrutura completa ou entendeu errado.
